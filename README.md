# Expense Tracker (Python)

A command-line based expense tracker built using Python that allows users to
record daily expenses, analyze category-wise spending, and visualize expense
distribution using pie charts.

## Features
- Add expenses with predefined or custom categories
- Persistent storage using CSV files
- Category-wise and total expense summary
- Data normalization and error handling
- Category-wise pie chart visualization using Matplotlib

## Tech Stack
- Python
- CSV File Handling
- Datetime Module
- Matplotlib

## How to Run
1. Clone the repository
2. Run `expense_tracker.py`
3. Use the menu to add expenses or view summaries and charts
